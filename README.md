# WuTang Helpdesk Lab

## Overview

This repository documents structured 1st Line / Helpdesk IT support lab scenarios based on real-world MSP tasks.

The focus is on practical troubleshooting, Active Directory administration, domain environments, shared drive management, and Microsoft 365 support.

## Objectives

- Strengthen 1st Line troubleshooting methodology
- Simulate real helpdesk ticket scenarios
- Improve clarity in technical communication
- Prepare for MSP-style technical interviews

## Lab Environment

- Windows Server (Domain Controller)
- Windows 10 Client VM
- Active Directory
- Group Policy
- File Shares & NTFS Permissions
- Microsoft 365 (where applicable)

---

Each lab entry includes:

- Scenario
- Objective
- Steps Taken
- Commands Used
- Result
- Key Learning

This lab is part of my structured preparation for a 1st Line IT Support / Helpdesk role.
